Docker学习(二)例子练习

部署Nginx

  • 搜索镜像去docker hub上
  • 下载镜像 docker pull nginx
  • docker run -d 后台运行 —name nginx01 -p 10024:80

-p 暴露端口的概念

思考：每次改动nginx配置文件，都需要进入容器内部，十分麻烦

可以在容器外部提供一个映射路径，达到在容器修改文件名，内部容器就可以自动修改。

这个技术是 -v 数据卷技术
